## 📖 專案說明

本專案 fork 自 [BugFor-Pings/CN_Sqlmap](https://github.com/BugFor-Pings/CN_Sqlmap)（簡體中文版），並進行以下改進：

- ✨ **轉換為繁體中文** - 所有界面文字改為台灣常用的繁體中文
- 🎨 **優化排版** - 在所有中英文之間自動添加空格，提升閱讀體驗
- 📝 **改善可讀性** - 符合繁體中文排版規範

### 原始專案說明

原簡體中文版基於 sqlmap 1.9 版本進行漢化，主要漢化內容包括：

1. 主要錯誤訊息和提示訊息
2. 程式橫幅和描述
3. 添加了漢化標識和版本資訊

## ⚙️ 使用方法

與原版 sqlmap 使用方法相同：

```bash
python sqlmap.py -h    # 顯示基本說明
python sqlmap.py -hh   # 顯示完整說明
```

## ⚠️ 注意事項

- 本繁體版保留了原始的功能和邏輯，僅更改界面顯示語言與排版
- 所有原始開發者和版權資訊保持不變
- 如遇到問題，可參考原始的英文錯誤訊息或查閱官方文件
- 漢化內容僅限於使用者界面，不影響程式功能
